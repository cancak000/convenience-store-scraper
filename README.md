# コンビニ公式サイト スクレイピングツール

## 概要
ファミリーマートやローソンなどの公式サイトから商品情報やキャンペーン情報を自動取得するPythonスクレイピングツールです。

## 使用技術
- Python 3.x
- requests
- BeautifulSoup
- pandas

## 使い方
1. `python scrape_famima.py` を実行
2. `output/famima_items.csv` に結果が出力されます

## 注意
このツールは教育目的で作成されており、対象サイトの利用規約を尊重してご使用ください。
